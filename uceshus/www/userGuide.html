Question Setter User Guide

For question submission, please fill in the form fully beneath the map on the homepage. A step-by-step guide is provided below.

  1. To begin, click on the map the location you wish to have your question placed for the quiz. Clicking on the map will display the latitude/longitude position of the point you selected.
  2. For navigation purposes, you are able to track your location to guide you with a point selection if needed. Do this by clicking "Track Location" on the navigation menu.
  3. Once the desired point has been selected, you are required to fill in the form and enter the latitude and longitude from your selected point on the map.
  4. For form entry:
    • "Site Location" = Name of place/road of selected point
    • "Question" = Enter question for quiz here
    • "Answer1, 2, 3, 4" = Enter 4 possible answers, providing only ONE correct answer.
    • "Correct Answer" = Enter the answer number that is correct (e.g. if answer3 is correct, enter 3).
    • "Latitude, Longitude" = Enter Lat/Lon provided on selected point on map.
  5. When complete, submit the form by clicking the "Start Data Upload" button.
  6. An alert will show the site location entered, click OK and await the result displayed under the button.
    • An error can occur if a field is NULL. All fields in the form must be filled.
  7. If the form uploaded successfully, the message beneath the start upload button will change from "The result of the upload goes here" to "Question Uploaded Successfully".
  8. You are able to test if the question is uploaded by clicking the "Get Question Data" button on the navigation menu, which will display the location point of the question you uploaded with an onClick event that pops up the site location name you entered.

If these steps are achieved, you have successfully uploaded a question to the appData database, ready for use in the QuizMapper app!
